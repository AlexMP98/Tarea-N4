**1. OBJETIVOS**

_Objetivo General_

-Comprender los teoremas y metodos utilizados para la resolucion de diferentes tipos de circuitos electricos.

_Objetivos Específicos_

-Analizar circuitos que tienen mas de una fuente de corriente.
